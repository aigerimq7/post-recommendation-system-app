# Система рекомендаций постов для социальной сети

Проект по разработке веб-сервиса (API), который рекомендует пользователям персональные посты на основе их профиля, истории лайков и временного контекста.

## 🚀 Основные результаты
* **Метрика качества:** Достигнут ROC-AUC ~ **0.675** на отложенной выборке.
* **Стек технологий:** Python, CatBoost, FastAPI, PostgreSQL, SQLAlchemy, Pandas, NumPy, NLP (TF-IDF, лемматизация, KMeans).

## 📊 Этапы разработки

### 1. Анализ данных и генерация признаков (ETL)
* Очищен и лемматизирован текст постов с помощью `nltk`.
* Построены текстовые эмбеддинги (TF-IDF), посты кластеризованы методом `KMeans`.
* Рассчитаны исторические CTR для пользователей (`user_avg_ctr`) и постов (`post_ctr`).

### 2. Обучение моделей
* Обучен `CatBoostClassifier` на GPU (1000 итераций).
* Проведен отбор признаков: удаление тяжелых текстовых дистанций позволило снизить требования к памяти Feature Store без значимой потери качества модели.

### 3. Развертывание сервиса (FastAPI)
* Реализован эндпоинт `/post/recommendations/`, возвращающий ТОП-5 постов для конкретного пользователя.
